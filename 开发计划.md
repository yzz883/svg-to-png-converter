# SVG转PNG工具开发计划

## 项目概述
本项目是一个基于Rust后端的SVG转PNG转换工具，提供Web界面让用户可以方便地将SVG文件或SVG代码转换为PNG图片并下载。

## 技术栈
- **后端**: Rust + Actix-web
- **前端**: 纯HTML + CSS + JavaScript
- **功能**: SVG转PNG（前端实现）

## 功能特性
1. **文件上传**: 支持拖拽上传SVG文件
2. **代码粘贴**: 支持直接粘贴SVG代码
3. **尺寸自定义**: 可自定义输出PNG的宽度和高度
4. **实时预览**: 转换后立即显示PNG预览
5. **一键下载**: 点击按钮即可下载转换后的PNG图片
