# SVG转PNG工具开发日志

## 2026-04-15

### 项目创建
- **时间**: 2026-04-15 18:00
- **内容**: 创建SVG转PNG工具项目
- **技术选择**:
  - 后端选择Rust + Actix-web，因为性能优秀且类型安全
  - 前端使用纯HTML/CSS/JavaScript，无需额外构建工具
  - SVG转PNG在前端实现，使用Canvas API

### 开发过程

#### 1. 后端开发
- 使用`cargo init`初始化项目
- 添加`actix-web`和`actix-files`依赖
- 实现简单的Web服务器，监听8081端口
- 配置路由返回HTML页面

#### 2. 前端开发
- 设计采用深色主题，现代简约风格
- 使用Google Fonts的Outfit和JetBrains Mono字体
- 实现两种输入方式：文件上传和代码粘贴
- 使用Canvas API实现SVG转PNG功能
- 添加尺寸自定义和下载功能

#### 3. 遇到的问题及解决方案

**问题1**: 端口8080被占用
- **原因**: 系统中已有服务占用8080端口
- **解决**: 修改为8081端口

**问题2**: 编译时文件被锁定
- **原因**: 之前的进程未完全退出
- **解决**: 使用`taskkill`强制终止进程

**问题3**: 链接器错误
- **原因**: 可执行文件被占用
- **解决**: 终止占用进程后重新编译

### 项目习惯
1. **端口管理**: 优先使用8081端口，避免常见端口冲突
2. **进程管理**: 开发时注意及时终止旧进程，避免文件锁定
3. **代码风格**: 
   - Rust代码遵循标准命名规范
   - 前端代码使用现代ES6+语法
   - CSS使用CSS变量管理主题色
